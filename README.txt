Описание классов:

.models.LetterEnum:

        Класс, в котором хранится список букв, предназначенных для номеров. Так же в нем реализовано два метода, а именно
    randomLetter() (предназначенный для выдачи рандомной буквы) и nextLetter(char letter) (для выдачи следующей буквы, которая
    идет после значения буквы, переданной в метод)

.models.Number:

        Ключевой класс, объект (один, единственный т.к. Синглтон) которого будет хранить в себе значение номера.

.servlets.MainServlet:

        Сервлет, который будет отрабатывать самым первым при запуске проекта. Содержит приветствие и подсказки, как получить
    номер.

.servlets.NextServlet:

        Сервлет, который отвечает за обработку последовательных номеров (/number/next)

.servlets.RandomServlet:

        Сервлет, который отвечает за обработку случайных номеров (/number/random)

.utils.CreateNumber:

        Основной класс, который создает номера в зависимости от URLa.

.utils.OperWithNumber:

        Вспомогательный класс, для обработки и проверки чисел
